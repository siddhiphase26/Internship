## Task-2
# Login & Signup Pages

This project contains two simple web pages:

1. Signup Page
2. Login Page

Both pages are designed using only **HTML and CSS**. They use a clean card layout with a gradient background and centered form design.

---

## 1. Signup Page

### Purpose

The Signup page is used to create a new user account. Users enter their personal details and select their user type.

### Main Features

* Full Name input
* Email input
* Phone Number input
* Password input
* Confirm Password input
* User Type dropdown (Student / Admin)
* Register button
* Link to Login page

### Code Explanation (Main Parts)

* **<!DOCTYPE html>**
  Tells the browser that this is an HTML5 document.

* **<html>, <head>, <body>**
  `<html>` contains the whole page.
  `<head>` contains the title and CSS.
  `<body>` contains what is visible on the screen.

* **<title>Signup</title>**
  Shows the text "Signup" on the browser tab.

* **Body background and centering**
  A gradient background is applied and flexbox is used to center the signup form in the middle of the screen.

* **.card class**
  Creates the white box for the signup form. It has padding, rounded corners, and shadow.

* **<h2>Create Account</h2>**
  This is the heading of the signup form.

* **<form> tag**
  Holds all input fields and the Register button. It is used to collect user information.

* **<label> tag**
  Displays the name of each input field (Full Name, Email, Phone, etc.).

* **<input> fields**
  Used to take user input such as name, email, phone number, and passwords.
  The `required` attribute forces the user to fill the field.

* **<select> dropdown**
  Used to select the user type: Student or Admin.

* **Register button**
  Submits the signup form.

* **Login link**
  Redirects the user to the Login page.

---

## 2. Login Page

### Purpose

The Login page is used by existing users to log into their account using their username and password.

### Main Features

* Username input
* Password input
* Login button
* Link to Signup page
* Forgot Password link

### Code Explanation (Main Parts)

# <!DOCTYPE html>
  Defines the document as HTML5.

# <html>, <head>, <body>
  `<html>` contains the whole page.
  `<head>` contains the title and CSS.
  `<body>` contains all visible elements.

# <title>Login</title>
  Displays "Login" on the browser tab.

# Body background and centering
  Gradient background is used and the login box is centered using flexbox.

# .card class
  Creates the white login box with padding, shadow, and rounded corners.

# <h2>User Login</h2>
  Heading of the login form.

# <form> tag
  Contains username field, password field, and the Login button.

# <label> tag
  Shows the name of each field (Username and Password).

# <input> fields
  Username is taken using text type.
  Password is hidden using password type.
  `required` ensures the user must fill the fields.

# Login button
  Submits the login form.

# Signup and Forgot Password links
  These allow navigation to the signup page and password recovery.

* The Signup page shows a centered card with all registration fields and a Register button.
* The Login page shows a centered card with username and password fields and a Login button.
* Both pages use the same background theme for a consistent look.
