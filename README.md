## Task-2
# Login & Signup Page Project
This project contains two simple web pages made using HTML and CSS:
•	Signup Page
•	Login Page
Both pages have a clean design with a centered form and gradient background.
________________________________________
# Signup Page
# Purpose
Used to create a new user account.
Fields
•	Full Name
•	Email
•	Phone Number
•	Password
•	Confirm Password
•	User Type (Student / Admin)
# Main Points
•	Uses a form to collect user details
•	Labels show what to enter in each box
•	Register button submits the form
•	Login link takes the user to the Login page

# Login Page
# Purpose
Used by existing users to log in.
Fields
•	Username
•	Password
# Main Points
•	Uses a form to take login details
•	Password is hidden while typing
•	Login button submits the form
•	Signup link opens the Signup page
•	Forgot Password link is also given

# Output
•	Signup page shows a registration form in the center
•	Login page shows a login form in the center
•	Both pages use the same background style

# Technologies Used
•	HTML
•	CSS

# How to Run
1.	Save the files as SignUp.html and Login.html
2.	Open any file in a web browser
3.	Use links to move between pages
