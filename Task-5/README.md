# IT Solution Technology– Static Website

##  Project Overview

This project is a **static website** built using **HTML and CSS only** for an IT company named IT Solution Technology. The website introduces the company and allows users to contact it through a simple form.

The project includes **three pages**:

* Home Page
* About Us Page
* Contact Us Page

A **common navigation bar (navbar)** and **footer** are reused across all pages to maintain consistency.

---

##  Project Structure

```
project/
├── index.html      (Home Page)
├── about.html      (About Us Page)
├── contact.html    (Contact Us Page)
└── style.css       (Single external CSS file)
```

---

##  Technologies Used

* **HTML5** (Semantic elements)
* **CSS3** (Flexbox, Grid, Responsive Design)

No JavaScript or backend is used.

---

##  Common Components

###  Navbar

* Appears on all pages
* Links placed on the **left side**
* Contains:

  * Home
  * About
  * Contact
* Company name displayed on the right side

###  Footer

* Appears on all pages
* Displays copyright information

---

##  Home Page (`index.html`)

### Sections Included:

1. **Hero Section**

   * Company tagline
   * Brief introduction

2. **Services Section**

   * Web Development
   * Software Solutions
   * IT Consulting

### Purpose:

To welcome users and give a quick overview of services offered by the company.

---

##  About Us Page (`about.html`)

### Sections Included:

* Who We Are
* Our Mission
* Our Vision

### Purpose:

To provide detailed information about the company and its goals.

---

##  Contact Us Page (`contact.html`)

### Form Fields:

* First Name
* Last Name
* Email
* Phone Number
* Message (textarea)

### Purpose:

To allow users to get in touch with the company.
