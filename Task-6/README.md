#  Signup / Registration Form

A modern, responsive **Signup / Registration Form** built using **HTML, CSS, and JavaScript**.  
This project includes **client-side validation**, **password strength checking**, and a **clean UI design**.

---

##  Features

- User-friendly signup form
-  First Name & Last Name validation
-  Phone number validation (10 digits)
-  Email format validation
-  Strong password rules
-  Password strength meter (Weak / Medium / Strong)
-  Show / Hide password option
-  Confirm password matching
-  User type selection (Student / Admin)
-  Responsive design (Mobile-friendly)
-  Clean and modern UI with gradient theme

##  Form Fields

| Field Name        | Validation Rules |
|------------------|------------------|
| First Name        | Required |
| Last Name         | Required |
| Phone Number      | Exactly 10 digits |
| Email             | Valid email format |
| Password          | Minimum 8 characters, 1 uppercase letter, 1 special symbol |
| Confirm Password  | Must match password |
| User Type         | Required (Student/Admin) |

---

##  Password Rules

The password must:
- Be at least **8 characters long**
- Contain **1 uppercase letter**
- Contain **1 special symbol**
- (For strong password) contain **1 number**
